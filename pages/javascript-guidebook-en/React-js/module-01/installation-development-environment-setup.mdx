## âš¡ Creating a New React Project with Vite

Vite makes starting a React project super fast. Here's how:

1. Open your terminal and run:

```bash
npm create vite@latest my-react-app -- --template react
```

2. Move into your project folder:

```bash
cd my-react-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

<Callout type="success">Your app will be live at <code>http://localhost:5173</code>! ğŸˆ</Callout>
