# React Development Environment Setup

In this section, we'll learn how to set up a modern React development environment using Vite. Vite is a build tool that provides a faster and leaner development experience for modern web projects.

## Prerequisites

Before we begin, make sure you have the following installed on your system:

- [Node.js](https://nodejs.org/) (version 16 or higher)
- A code editor (we recommend [VS Code](https://code.visualstudio.com/))
- npm or yarn package manager

## Creating a New React Project with Vite

Vite provides a quick way to scaffold a new React project. Follow these steps:

1. Open your terminal and run the following command:

```bash
npm create vite@latest my-react-app -- --template react
```

2. Navigate to your project directory:

```bash
cd my-react-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

Your React application will be available at `http://localhost:5173`.

## Project Structure

After creating your project, you'll see the following structure:

```
my-react-app/
├── node_modules/
├── public/
├── src/
│   ├── assets/
│   ├── App.css
│   ├── App.jsx
│   ├── index.css
│   └── main.jsx
├── .gitignore
├── index.html
├── package.json
├── vite.config.js
└── README.md
```

## Key Files Explained

- `src/main.jsx`: The entry point of your React application
- `src/App.jsx`: The main App component
- `vite.config.js`: Vite configuration file
- `index.html`: The HTML template

## Recommended VS Code Settings

You can copy these settings and adjust them according to your current configurations. Add them to your `.vscode/settings.json` file for best results:

```json
{
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": "explicit",
    "source.fixAll.tslint": "explicit",
    "source.organizeImports": "explicit"
  },
  "eslint.run": "onSave",
  "emmet.includeLanguages": {
    "javascript": "javascriptreact"
  },
  "path-autocomplete.extensionOnImport": true,
  "path-autocomplete.excludedItems": {
    "*/.js": {
      "when": "**"
    },
    "*/.jsx": {
      "when": "**"
    }
  },
  "javascript.validate.enable": false,
  "typescript.validate.enable": false
}
```
