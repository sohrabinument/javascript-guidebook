# ✨ Writing Markup with JSX

Welcome! If you want to build beautiful UIs with React, JSX is your new best friend. Let's make learning JSX fun and easy! 🚀

---

## 🤔 What is JSX?

JSX (JavaScript XML) is a syntax extension for JavaScript that lets you write HTML-like markup inside a JavaScript file. It makes writing UI components in React more intuitive and readable.

<Callout type="info">
  Think of JSX as a bridge between JavaScript and HTML—best of both worlds!
</Callout>

---

## 💡 Why use JSX in React?

- **Combines markup and logic:** Write markup and rendering logic together in components, keeping related code in one place.
- **Familiar syntax:** JSX looks similar to HTML, making it easier for web developers to learn and use.
- **Dynamic content:** Embed JavaScript expressions inside JSX using curly braces `{}`.

---

## 🔍 How is JSX different from HTML?

- **JSX is not HTML:** It's a syntax extension that gets compiled to JavaScript.
- **Stricter rules:** JSX requires all tags to be closed, and you must use camelCase for most attribute names (e.g., `className` instead of `class`).

<Callout type="warning">
  Don't forget: `class` becomes `className`, and all tags must be closed!
</Callout>

---

## 📝 Basic Example

```jsx
function Welcome() {
  return <h1>Hello, world!</h1>;
}
```

---

## 🧩 Embedding JavaScript in JSX

You can use curly braces `{}` to embed JavaScript expressions:

```jsx
const name = "Hedy Lamarr";
const element = <h1>Hello, {name}!</h1>;
```

---

## 📏 Common JSX Rules

1. **Return a single root element:**  
   Wrap multiple elements in a parent tag or use a React Fragment (`<>...</>`).

   ```jsx
   return (
     <>
       <h1>Title</h1>
       <p>Description</p>
     </>
   );
   ```

2. **Close all tags:**  
   Self-closing tags like `<img />` must be closed.

   ```jsx
   <img src="image.png" alt="desc" />
   ```

3. **Use camelCase for attributes:**  
   - `className` instead of `class`
   - `htmlFor` instead of `for`

   ```jsx
   <div className="container"></div>
   ```

---

## 🔄 Converting HTML to JSX

- Change `class` to `className`
- Close all tags
- Use camelCase for attributes

**Example:**

HTML:
```html
<div class="intro">
  <h1>Welcome!</h1>
</div>
```

JSX:
```jsx
<div className="intro">
  <h1>Welcome!</h1>
</div>
```

<Callout type="info">
  If you have a lot of HTML to convert, use an [online JSX converter](https://transform.tools/html-to-jsx).
</Callout>
