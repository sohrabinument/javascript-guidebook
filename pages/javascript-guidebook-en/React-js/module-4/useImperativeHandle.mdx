**When using `forwardRef`, we forward all access to our component to the parent component. But what if we want to forward the reference, but also specify exactly what the parent component can do with that reference? In other words, we want to limit the access from the child to the parent component. If we want to do this, we'll need to use a built-in React hook called `useImperativeHandle`. We can do this by following the steps below.**

<Steps>
### The child component needs to take a `ref` for the component's NODE and use the child component's `ref`.

```jsx {4,8}
import { forwardRef, useRef } from "react";

const MyInput = forwardRef((props, ref) => {
    const realInputRef = useRef(null);
    return (
        <input
            {...props}
            ref={realInputRef}
        />
    );
});
```

### Use the `useImperativeHandle` hook to specify which functionalities we want to give access to.

```jsx {7-11}
import { forwardRef, useRef, useImperativeHandle } from "react";

const MyInput = forwardRef((props, ref) => {
    const realInputRef = useRef(null);

    // Only expose focus and nothing else
    useImperativeHandle(ref, () => ({
        focus() {
            realInputRef.current.focus();
        },
    }));

    return (
        <input
            {...props}
            ref={realInputRef}
        />
    );
});
```

</Steps>
