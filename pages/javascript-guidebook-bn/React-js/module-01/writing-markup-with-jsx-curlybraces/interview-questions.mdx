# JSX কার্লি ব্রেসে ইন্টারভিউ প্রশ্ন ও উত্তর

## বেসিক

**প্রশ্ন ১: JSX-এ কার্লি ব্রেস `{}` কেন ব্যবহার করা হয়?**  
উত্তর: কার্লি ব্রেস ব্যবহার করে JSX মার্কআপের মধ্যে জাভাস্ক্রিপ্ট এক্সপ্রেশন, ভ্যারিয়েবল বা ফাংশন কল এম্বেড করা যায়।

**প্রশ্ন ২: JSX-এ কার্লি ব্রেস কোথায় ব্যবহার করা যায়?**  
উত্তর: ট্যাগ কনটেন্টে (যেমন, `<h1>{name}</h1>`) এবং অ্যাট্রিবিউট ভ্যালু হিসেবে (যেমন, `<img src={avatar} />`)।

---

## ইন্টারমিডিয়েট

**প্রশ্ন ৩: JSX-এ `src="avatar.png"` এবং `src={avatar}` এর মধ্যে পার্থক্য কী?**  
উত্তর: `src="avatar.png"` স্ট্রিং "avatar.png" পাস করে, আর `src={avatar}` ভ্যারিয়েবল `avatar`-এর ভ্যালু পাস করে।

**প্রশ্ন ৪: JSX-এ ইনলাইন স্টাইলের জন্য অবজেক্ট কীভাবে ব্যবহার করবেন?**  
উত্তর: ডাবল কার্লি ব্রেস ব্যবহার করতে হবে: `<div style={{ color: 'red' }}></div>`। বাইরের ব্রেস JSX-এর জন্য, ভেতরের ব্রেস জাভাস্ক্রিপ্ট অবজেক্টের জন্য।

## অ্যাডভান্সড / সিনারিও-ভিত্তিক

**প্রশ্ন ৫: নিচের JSX কি বৈধ? কেন?**

```jsx
<{tag}>Hello</{tag}>
```

উত্তর: অবৈধ। JSX-এ কার্লি ব্রেস দিয়ে ট্যাগ নাম ব্যবহার করা যায় না।

**প্রশ্ন ৬: JSX-এ `<img src="{avatar}" />` লিখলে কী হবে?**  
উত্তর: এটি স্ট্রিং `"{avatar}"` পাস করবে, ভ্যারিয়েবল নয়। সঠিকভাবে ব্যবহার করতে হলে `src={avatar}` লিখতে হবে।

**প্রশ্ন ৭: JSX-এ কার্লি ব্রেসের মধ্যে ফাংশন কল ব্যবহার করা যায় কি? উদাহরণ দিন।**  
উত্তর: হ্যাঁ। উদাহরণ: `<h1>{formatName(user)}</h1>`

\*\*প্রশ্ন ৮: নিচের কোডের আউটপুট কী হবে?

```jsx
const count = 5;
return <p>{count + 1}</p>;
```

উত্তর: এটি `<p>6</p>` রেন্ডার করবে, কারণ কার্লি ব্রেসের মধ্যে এক্সপ্রেশনটি ইভ্যালুয়েট হয়।
