# JSX-এ কার্লি ব্রেসে জাভাস্ক্রিপ্ট

## পরিচিতি

JSX আপনাকে জাভাস্ক্রিপ্ট ফাইলে HTML-এর মতো মার্কআপ লিখতে দেয়। অনেক সময়, আপনাকে মার্কআপের মধ্যে জাভাস্ক্রিপ্ট লজিক বা ডাইনামিক ভ্যালু ব্যবহার করতে হয়। এই ক্ষেত্রে, JSX-এ কার্লি ব্রেস `{}` ব্যবহার করে আপনি সরাসরি জাভাস্ক্রিপ্ট ব্যবহার করতে পারেন।

---

## JSX-এ কার্লি ব্রেস কোথায় ব্যবহার করা যায়?

1. **ট্যাগ কনটেন্টে:**
   ```jsx
   <h1>{name} is a great teacher</h1>
   ```
   এখানে `{name}` একটি জাভাস্ক্রিপ্ট ভ্যারিয়েবল।

2. **অ্যাট্রিবিউট ভ্যালু হিসেবে:**
   ```jsx
   <img src={avatarUrl} alt={description} />
   ```
   এখানে কার্লি ব্রেসের মধ্যে যেকোনো জাভাস্ক্রিপ্ট এক্সপ্রেশন ব্যবহার করা যায়।

---

## কার্লি ব্রেসের মধ্যে কী রাখা যায়?
- ভ্যারিয়েবল
- ফাংশন কল
- এক্সপ্রেশন (যেমন, `{count + 1}`)
- অবজেক্ট (যেমন, ইনলাইন স্টাইলের জন্য)

**উদাহরণ:**
```jsx
const name = 'Hedy Lamarr';
const avatar = 'https://i.imgur.com/yXOvdOSs.jpg';
const style = { color: 'blue', backgroundColor: 'lightgray' };

function Profile() {
  return (
    <div style={style}>
      <h1>{name}'s Profile</h1>
      <img src={avatar} alt={name} />
    </div>
  );
}
```

---

## JSX-এ কার্লি ব্রেস ব্যবহারের নিয়ম

1. **শুধুমাত্র দুই জায়গায় কার্লি ব্রেস ব্যবহার করা যায়:**
   - JSX ট্যাগের কনটেন্টে: `<h1>{name}</h1>`
   - অ্যাট্রিবিউট ভ্যালু হিসেবে: `<img src={avatar} />`

2. **কার্লি ব্রেস দিয়ে ট্যাগ নাম ব্যবহার করা যায় না:**
   - ❌ `<{tag}>Hello</{tag}>` এটি অবৈধ।

3. **স্ট্রিং অ্যাট্রিবিউট বনাম জাভাস্ক্রিপ্ট এক্সপ্রেশন:**
   - `src="avatar.png"` স্ট্রিং হিসেবে পাস হয়।
   - `src={avatar}` ভ্যারিয়েবলের ভ্যালু পাস হয়।

4. **অবজেক্ট (যেমন ইনলাইন স্টাইল) এর জন্য ডাবল কার্লি ব্রেস ব্যবহার করুন:**
   - `<div style={{ color: 'red' }}></div>`
   - বাইরের `{}` JSX-এর জন্য, ভেতরের `{}` জাভাস্ক্রিপ্ট অবজেক্টের জন্য।

---

## সাধারণ ভুল
- ট্যাগ নামের জন্য কার্লি ব্রেস ব্যবহার করা।
- কোটেশন ও কার্লি ব্রেস একসাথে ব্যবহার করা: `src="{avatar}"` (এটি স্ট্রিং `"{avatar}"` পাস করে, ভ্যারিয়েবল নয়)।
- ট্যাগ ক্লোজ করতে ভুলে যাওয়া।
