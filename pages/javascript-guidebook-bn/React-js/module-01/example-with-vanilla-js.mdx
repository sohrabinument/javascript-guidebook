# Vanilla JavaScript এর উদাহরণ

Vanilla JavaScript ব্যবহার করে একটি সাধারণ প্রোডাক্ট কার্ট তৈরি করা যাক। এই উদাহরণটি দেখাবে কিভাবে DOM ম্যানিপুলেশন এবং ইভেন্ট হ্যান্ডলিং করা হয়।

## প্রোডাক্ট কার্টের HTML স্ট্রাকচার

```html
<div class="cart">
  <h2>আপনার কার্ট</h2>
  <div class="cart-items">
    <!-- কার্ট আইটেমগুলি এখানে যোগ করা হবে -->
  </div>
  <div class="cart-total">
    <p>মোট: <span class="total-price">৳0</span></p>
  </div>
</div>

<div class="products">
  <div class="product" data-id="1" data-price="100">
    <h3>প্রোডাক্ট 1</h3>
    <p>৳100</p>
    <button class="add-to-cart">কার্টে যোগ করুন</button>
  </div>
  <div class="product" data-id="2" data-price="200">
    <h3>প্রোডাক্ট 2</h3>
    <p>৳200</p>
    <button class="add-to-cart">কার্টে যোগ করুন</button>
  </div>
</div>
```

## JavaScript কোড

```javascript
// কার্ট স্টেট
let cart = [];

// DOM এলিমেন্টগুলি
const cartItemsContainer = document.querySelector(".cart-items");
const totalPriceElement = document.querySelector(".total-price");
const addToCartButtons = document.querySelectorAll(".add-to-cart");

// কার্টে আইটেম যোগ করার ফাংশন
function addToCart(productId, price) {
  // কার্টে আইটেম যোগ করুন
  cart.push({ id: productId, price: price });

  // UI আপডেট করুন
  updateCartUI();
}

// কার্ট থেকে আইটেম মুছে ফেলার ফাংশন
function removeFromCart(productId) {
  // কার্ট থেকে আইটেম মুছে ফেলুন
  cart = cart.filter((item) => item.id !== productId);

  // UI আপডেট করুন
  updateCartUI();
}

// কার্ট UI আপডেট করার ফাংশন
function updateCartUI() {
  // কার্ট আইটেম কন্টেইনার খালি করুন
  cartItemsContainer.innerHTML = "";

  // প্রতিটি কার্ট আইটেমের জন্য HTML তৈরি করুন
  cart.forEach((item) => {
    const cartItem = document.createElement("div");
    cartItem.className = "cart-item";
    cartItem.innerHTML = `
      <span>প্রোডাক্ট ${item.id}</span>
      <span>৳${item.price}</span>
      <button onclick="removeFromCart(${item.id})">মুছে ফেলুন</button>
    `;
    cartItemsContainer.appendChild(cartItem);
  });

  // মোট মূল্য গণনা করুন
  const total = cart.reduce((sum, item) => sum + item.price, 0);
  totalPriceElement.textContent = `৳${total}`;
}

// ইভেন্ট লিসেনার যোগ করুন
addToCartButtons.forEach((button) => {
  button.addEventListener("click", () => {
    const product = button.closest(".product");
    const productId = product.dataset.id;
    const price = parseInt(product.dataset.price);
    addToCart(productId, price);
  });
});
```

## এই উদাহরণে কি ঘটছে?

1. **স্টেট ম্যানেজমেন্ট**:

   - একটি `cart` অ্যারে ব্যবহার করে কার্টের স্টেট ট্র্যাক করা হয়
   - প্রতিটি আইটেম যোগ বা মুছে ফেলার সময় স্টেট আপডেট করা হয়

2. **DOM ম্যানিপুলেশন**:

   - `innerHTML` এবং `createElement` ব্যবহার করে DOM আপডেট করা হয়
   - প্রতিটি পরিবর্তনের জন্য UI ম্যানুয়ালি আপডেট করতে হয়

3. **ইভেন্ট হ্যান্ডলিং**:

   - `addEventListener` ব্যবহার করে বাটন ক্লিক হ্যান্ডল করা হয়
   - প্রতিটি প্রোডাক্টের জন্য আলাদা ইভেন্ট লিসেনার যোগ করা হয়

4. **UI আপডেট**:
   - প্রতিটি পরিবর্তনের জন্য পুরো কার্ট UI আপডেট করতে হয়
   - মোট মূল্য প্রতিটি পরিবর্তনের সময় পুনরায় গণনা করা হয়

## এই পদ্ধতির সীমাবদ্ধতা

1. **জটিলতা**:

   - বড় অ্যাপ্লিকেশনের জন্য কোড জটিল হয়ে উঠতে পারে
   - স্টেট ম্যানেজমেন্ট কঠিন হয়ে পড়ে

2. **পারফরম্যান্স**:

   - প্রতিটি পরিবর্তনের জন্য পুরো UI আপডেট করতে হয়
   - অপ্রয়োজনীয় DOM অপারেশন হতে পারে

3. **রক্ষণাবেক্ষণ**:
   - কোড বাড়ার সাথে সাথে রক্ষণাবেক্ষণ কঠিন হয়ে পড়ে
   - বাগ খুঁজে পাওয়া কঠিন হতে পারে
