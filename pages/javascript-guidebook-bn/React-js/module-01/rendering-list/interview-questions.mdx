# React-এ লিস্ট রেন্ডারিং: ইন্টারভিউ প্রশ্ন ও উত্তর

1. **React-এ কিভাবে একটি লিস্ট রেন্ডার করবেন?**  
   **উত্তর:** React-এ লিস্ট রেন্ডার করতে JavaScript-এর `map()` ফাংশন ব্যবহার করে ডেটার অ্যারে থেকে JSX এলিমেন্টের অ্যারে তৈরি করা হয়। যেমন:

   ```jsx
   const items = ["Apple", "Banana", "Cherry"];
   const listItems = items.map((item) => <li>{item}</li>);
   return <ul>{listItems}</ul>;
   ```

2. **লিস্ট রেন্ডার করার সময় `key` prop কেন গুরুত্বপূর্ণ?**  
   **উত্তর:** `key` prop React-কে জানায় কোন আইটেম পরিবর্তিত হয়েছে, যোগ হয়েছে বা মুছে গেছে। এতে React শুধু পরিবর্তিত এলিমেন্টগুলো আপডেট করতে পারে, ফলে পারফরম্যান্স বাড়ে এবং বাগ কমে।

3. **অ্যারে ইনডেক্সকে key হিসেবে ব্যবহার করলে কী সমস্যা হতে পারে?**  
   **উত্তর:** অ্যারে ইনডেক্সকে key হিসেবে ব্যবহার করলে লিস্টে আইটেম যোগ/বিয়োগ বা অর্ডার পরিবর্তন হলে React সঠিকভাবে আইটেম ট্র্যাক করতে পারে না। এতে অপ্রত্যাশিত আচরণ, কম্পোনেন্ট স্টেট হারানো বা অপ্রয়োজনীয় রি-রেন্ডার হতে পারে।

4. **React-এ রেন্ডার করার আগে কিভাবে লিস্ট ফিল্টার করবেন?**  
   **উত্তর:** JavaScript-এর `filter()` মেথড ব্যবহার করে নির্দিষ্ট আইটেমের নতুন অ্যারে তৈরি করে, তারপর `map()` দিয়ে রেন্ডার করা যায়। যেমন:

   ```jsx
   const fruits = ["Apple", "Banana", "Cherry"];
   const filtered = fruits.filter((fruit) => fruit.startsWith("B"));
   const listItems = filtered.map((fruit) => <li>{fruit}</li>);
   return <ul>{listItems}</ul>;
   ```

5. **একটি লিস্টের প্রতিটি আইটেমের জন্য একাধিক এলিমেন্ট কিভাবে রেন্ডার করবেন?**  
   **উত্তর:** প্রতিটি আইটেমের জন্য একাধিক এলিমেন্ট রেন্ডার করতে `<React.Fragment>` (বা সংক্ষিপ্ত `<>...</>`) ব্যবহার করে এবং key দিতে পারেন:

   ```jsx
   import { Fragment } from "react";
   const listItems = people.map((person) => (
     <Fragment key={person.id}>
       <h2>{person.name}</h2>
       <p>{person.profession}</p>
     </Fragment>
   ));
   ```

6. **লিস্ট রেন্ডার করার সময় key না দিলে কী হয়?**  
   **উত্তর:** React কনসোলে ওয়ার্নিং দেখাবে। ইউনিক key ছাড়া React সঠিকভাবে আইটেম ট্র্যাক করতে পারে না, ফলে অপ্রয়োজনীয় আপডেট ও বাগ হতে পারে।

7. **যেসব লিস্টে আইটেম যোগ/বিয়োগ হতে পারে, সেগুলো কিভাবে হ্যান্ডেল করবেন?**  
   **উত্তর:** প্রতিটি লিস্ট আইটেমের জন্য ডেটা থেকে ইউনিক ও স্থিতিশীল key (যেমন ID) ব্যবহার করুন। এতে React সঠিকভাবে লিস্ট আপডেট করতে পারে।
