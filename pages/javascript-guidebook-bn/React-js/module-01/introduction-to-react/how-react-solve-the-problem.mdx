# React কিভাবে সমস্যা সমাধান করে

আগের বিভাগে আমরা Vanilla JavaScript এর সমস্যাগুলি দেখেছি। এখন আমরা দেখব কিভাবে React এই সমস্যাগুলি সমাধান করে।

## 1. DOM ম্যানিপুলেশন সমস্যা সমাধান

Vanilla JavaScript এ:

```javascript
// ম্যানুয়াল DOM ম্যানিপুলেশন
const element = document.getElementById("myElement");
element.textContent = "Hello World";
```

React এ:

```jsx
// ডিক্লারেটিভ UI
function Greeting() {
  return <h1>Hello World</h1>;
}
```

**কিভাবে সমাধান করে:**

- UI কে স্টেটের ফাংশন হিসেবে প্রকাশ করে
- DOM আপডেট React এর দায়িত্ব
- ডেভেলপার শুধু UI ডেসক্রাইব করে
- অটোমেটিক DOM আপডেট

## 2. স্টেট ম্যানেজমেন্ট সমস্যা সমাধান

Vanilla JavaScript এ:

```javascript
// গ্লোবাল স্টেট
let user = {
  name: "",
  cart: [],
};

function updateUser(newData) {
  user = { ...user, ...newData };
  updateUI();
}
```

React এ:

```jsx
// স্টেট হুক ব্যবহার
function UserProfile() {
  const [user, setUser] = useState({
    name: "",
    cart: [],
  });

  const updateUser = (newData) => {
    setUser((prev) => ({ ...prev, ...newData }));
  };

  return (
    <div>
      <h1>{user.name}</h1>
      <Cart items={user.cart} />
    </div>
  );
}
```

**কিভাবে সমাধান করে:**

- স্টেট ম্যানেজমেন্ট বিল্ট-ইন
- UI অটোমেটিক আপডেট
- স্টেট পরিবর্তন ট্র্যাক করা সহজ
- কম্পোনেন্ট-লেভেল স্টেট

## 3. ইভেন্ট হ্যান্ডলিং সমস্যা সমাধান

Vanilla JavaScript এ:

```javascript
// মাল্টিপল ইভেন্ট লিসেনার
button1.addEventListener("click", handleClick1);
button2.addEventListener("click", handleClick2);
// ... আরও অনেক
```

React এ:

```jsx
// ডিক্লারেটিভ ইভেন্ট হ্যান্ডলিং
function Buttons() {
  return (
    <div>
      <button onClick={handleClick1}>Button 1</button>
      <button onClick={handleClick2}>Button 2</button>
    </div>
  );
}
```

**কিভাবে সমাধান করে:**

- ইভেন্ট হ্যান্ডলিং ডিক্লারেটিভ
- মেমরি ম্যানেজমেন্ট অটোমেটিক
- ইভেন্ট ক্লিনআপ সহজ
- কম্পোনেন্ট আনমাউন্টে অটোমেটিক ক্লিনআপ

## 4. কোড অর্গানাইজেশন সমস্যা সমাধান

Vanilla JavaScript এ:

```javascript
// সব কোড একটি ফাইলে
const user = {
  /* ... */
};
const cart = {
  /* ... */
};
const products = {
  /* ... */
};
```

React এ:

```jsx
// মডুলার কম্পোনেন্ট
function UserProfile() {
  return <div>...</div>;
}

function Cart() {
  return <div>...</div>;
}

function Products() {
  return <div>...</div>;
}
```

**কিভাবে সমাধান করে:**

- কম্পোনেন্ট-ভিত্তিক আর্কিটেকচার
- কোড মডুলারাইজেশন
- রিইউজেবল কম্পোনেন্ট
- ক্লিন কোড স্ট্রাকচার

## 5. পারফরম্যান্স সমস্যা সমাধান

Vanilla JavaScript এ:

```javascript
// অদক্ষ DOM অপারেশন
function updateList(items) {
  const list = document.getElementById("list");
  list.innerHTML = "";
  items.forEach((item) => {
    const li = document.createElement("li");
    li.textContent = item;
    list.appendChild(li);
  });
}
```

React এ:

```jsx
// ভার্চুয়াল DOM ব্যবহার
function List({ items }) {
  return (
    <ul>
      {items.map((item) => (
        <li key={item.id}>{item.text}</li>
      ))}
    </ul>
  );
}
```

**কিভাবে সমাধান করে:**

- ভার্চুয়াল DOM ব্যবহার
- মিনিমাল DOM আপডেট
- রেন্ডার অপটিমাইজেশন
- পারফরম্যান্স বিল্ট-ইন

## 6. স্কেলিং সমস্যা সমাধান

Vanilla JavaScript এ:

```javascript
// স্টেট ডিপেন্ডেন্সি
function updateUI() {
  if (user.loggedIn) {
    updateHeader();
    updateSidebar();
    updateContent();
    updateFooter();
  }
  // ... আরও অনেক কন্ডিশন
}
```

React এ:

```jsx
// কম্পোনেন্ট-ভিত্তিক আর্কিটেকচার
function App() {
  return (
    <>
      <Header />
      <Sidebar />
      <Content />
      <Footer />
    </>
  );
}
```

**কিভাবে সমাধান করে:**

- কম্পোনেন্ট আইসোলেশন
- স্টেট লোকালাইজেশন
- রিইউজেবল কম্পোনেন্ট
- মডুলার ডেভেলপমেন্ট

## বাস্তব-বিশ্বের উদাহরণ: প্রোডাক্ট কার্ট

Vanilla JavaScript এ প্রোডাক্ট কার্ট:

```javascript
// জটিল স্টেট ম্যানেজমেন্ট
let cart = [];
let total = 0;

function addToCart(product) {
  cart.push(product);
  total += product.price;
  updateCartUI();
  updateTotal();
  updateCheckoutButton();
}
```

React এ প্রোডাক্ট কার্ট:

```jsx
// সিম্পল স্টেট ম্যানেজমেন্ট
function Cart() {
  const [cart, setCart] = useState([]);

  const addToCart = (product) => {
    setCart((prev) => [...prev, product]);
  };

  const total = cart.reduce((sum, item) => sum + item.price, 0);

  return (
    <div>
      <h2>আপনার কার্ট</h2>
      {cart.map((item) => (
        <CartItem key={item.id} item={item} />
      ))}
      <p>মোট: ৳{total}</p>
    </div>
  );
}
```

## উপসংহার

React Vanilla JavaScript এর সমস্যাগুলি সমাধান করে:

- ডিক্লারেটিভ UI প্রোগ্রামিং
- কম্পোনেন্ট-ভিত্তিক আর্কিটেকচার
- ভার্চুয়াল DOM
- স্টেট ম্যানেজমেন্ট
- পারফরম্যান্স অপটিমাইজেশন
- স্কেলেবল ডেভেলপমেন্ট

এই কারণেই React আধুনিক ওয়েব ডেভেলপমেন্টের জন্য জনপ্রিয় পছন্দ।
