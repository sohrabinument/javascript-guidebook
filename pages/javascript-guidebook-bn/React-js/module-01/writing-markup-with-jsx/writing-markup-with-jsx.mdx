# JSX দিয়ে মার্কআপ লেখা

## JSX কী?

JSX (JavaScript XML) হলো জাভাস্ক্রিপ্টের জন্য একটি সিনট্যাক্স এক্সটেনশন, যা আপনাকে জাভাস্ক্রিপ্ট ফাইলে HTML-এর মতো মার্কআপ লিখতে দেয়। এটি React-এ UI কম্পোনেন্ট লেখা আরও সহজ ও বোধগম্য করে তোলে।

## React-এ কেন JSX ব্যবহার করা হয়?

- **মার্কআপ ও লজিক একসাথে:** JSX ব্যবহার করে আপনি কম্পোনেন্টে মার্কআপ ও রেন্ডারিং লজিক একসাথে লিখতে পারেন, ফলে সম্পর্কিত কোড এক জায়গায় থাকে।
- **পরিচিত সিনট্যাক্স:** JSX দেখতে অনেকটা HTML-এর মতো, তাই ওয়েব ডেভেলপারদের জন্য শেখা সহজ।
- **ডাইনামিক কনটেন্ট:** JSX-এ কার্লি ব্রেস `{}` ব্যবহার করে জাভাস্ক্রিপ্ট এক্সপ্রেশন এম্বেড করা যায়।

## JSX ও HTML-এর পার্থক্য কী?

- **JSX, HTML নয়:** এটি একটি সিনট্যাক্স এক্সটেনশন, যা জাভাস্ক্রিপ্টে কম্পাইল হয়।
- **কঠোর নিয়ম:** JSX-এ সব ট্যাগ ক্লোজ করতে হয় এবং বেশিরভাগ অ্যাট্রিবিউটে camelCase ব্যবহার করতে হয় (যেমন, `class` এর পরিবর্তে `className`)।

## বেসিক উদাহরণ

```jsx
function Welcome() {
  return <h1>Hello, world!</h1>;
}
```

## JSX-এ জাভাস্ক্রিপ্ট এম্বেড করা

কার্লি ব্রেস `{}` ব্যবহার করে জাভাস্ক্রিপ্ট এক্সপ্রেশন এম্বেড করা যায়:

```jsx
const name = "Hedy Lamarr";
const element = <h1>Hello, {name}!</h1>;
```

## JSX-এর সাধারণ নিয়ম

1. **একটি রুট এলিমেন্ট রিটার্ন করুন:**  
   একাধিক এলিমেন্ট রিটার্ন করতে হলে একটি প্যারেন্ট ট্যাগ বা React Fragment (`<>...</>`) ব্যবহার করুন।

   ```jsx
   return (
     <>
       <h1>Title</h1>
       <p>Description</p>
     </>
   );
   ```

2. **সব ট্যাগ ক্লোজ করুন:**  
   `<img />` এর মতো সেল্ফ-ক্লোজিং ট্যাগ অবশ্যই ক্লোজ করতে হবে।

   ```jsx
   <img src="image.png" alt="desc" />
   ```

3. **অ্যাট্রিবিউটে camelCase ব্যবহার করুন:**  
   - `class` এর পরিবর্তে `className`
   - `for` এর পরিবর্তে `htmlFor`

   ```jsx
   <div className="container"></div>
   ```

## HTML থেকে JSX-এ রূপান্তর

- `class` কে `className` করুন
- সব ট্যাগ ক্লোজ করুন
- অ্যাট্রিবিউটে camelCase ব্যবহার করুন

**উদাহরণ:**

HTML:
```html
<div class="intro">
  <h1>Welcome!</h1>
</div>
```

JSX:
```jsx
<div className="intro">
  <h1>Welcome!</h1>
</div>
```
