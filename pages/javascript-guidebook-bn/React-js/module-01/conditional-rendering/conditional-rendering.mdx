# কন্ডিশনাল রেন্ডারিং ইন React

## পরিচিতি

React-এ কন্ডিশনাল রেন্ডারিং মানে হচ্ছে নির্দিষ্ট কোনো শর্ত অনুযায়ী বিভিন্ন UI এলিমেন্ট বা কম্পোনেন্ট দেখানো। এটি অনেকটা JavaScript-এর if/else-এর মতো, তবে এখানে JSX-এর মধ্যে ব্যবহার করা হয়।

## কন্ডিশনাল রেন্ডারিং করার সাধারণ উপায়

### ১. if/else স্টেটমেন্ট ব্যবহার

আপনি আপনার কম্পোনেন্টের ভিতরে if/else স্টেটমেন্ট ব্যবহার করে কী রিটার্ন করবেন তা নির্ধারণ করতে পারেন:

```jsx
function Greeting({ isLoggedIn }) {
  if (isLoggedIn) {
    return <h1>স্বাগতম!</h1>;
  }
  return <h1>অনুগ্রহ করে সাইন ইন করুন।</h1>;
}
```

### ২. টার্নারি অপারেটর ব্যবহার

টার্নারি অপারেটর একটি শর্তের উপর ভিত্তি করে কনটেন্ট রেন্ডার করার সংক্ষিপ্ত উপায়:

```jsx
function Greeting({ isLoggedIn }) {
  return (
    <div>
      {isLoggedIn ? <h1>স্বাগতম!</h1> : <h1>অনুগ্রহ করে সাইন ইন করুন।</h1>}
    </div>
  );
}
```

### ৩. লজিক্যাল AND (&&) অপারেটর ব্যবহার

শুধুমাত্র শর্ত সত্য হলে কিছু রেন্ডার করতে চাইলে `&&` অপারেটর ব্যবহার করতে পারেন:

```jsx
function Notification({ hasUnread }) {
  return (
    <div>
      {hasUnread && <p>আপনার অপঠিত মেসেজ আছে!</p>}
    </div>
  );
}
```

### ৪. ভ্যারিয়েবলে JSX অ্যাসাইন করা

আরও জটিল শর্তের জন্য, আপনি JSX-কে একটি ভ্যারিয়েবলে রেখে পরে রিটার্ন করতে পারেন:

```jsx
function UserStatus({ isOnline }) {
  let status;
  if (isOnline) {
    status = <span className="online">অনলাইনে</span>;
  } else {
    status = <span className="offline">অফলাইনে</span>;
  }
  return <div>ইউজার: {status}</div>;
}
```

## বেস্ট প্র্যাকটিস

- সহজ কন্ডিশনের জন্য টার্নারি অপারেটর ব্যবহার করুন।
- জটিল লজিকের জন্য if/else বা ভ্যারিয়েবল ব্যবহার করুন।
- গভীরভাবে নেস্টেড টার্নারি অপারেটর এড়িয়ে চলুন।

```jsx
function Item({ name, isPacked }) {
  return (
    <li className="item">
      {isPacked ? <del>{name + ' ✅'}</del> : name}
    </li>
  );
}
```
